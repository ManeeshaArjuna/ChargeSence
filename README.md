# ⚡ ChargeSence – Smart EV Charging System

## 📌 Overview

ChargeSence is a smart EV charging management system with AI-based charger recommendations, booking system, wallet payments, and queue management.

---

## 🏗️ Project Structure

* **Backend**: Django + Django REST Framework
* **Frontend**: React
* **Database**: SQLite / PostgreSQL

---

## 🚀 Features

* User Authentication (JWT)
* EV Charger Booking System
* AI-Based Charger Recommendation
* Google Maps Route Visualization
* Wallet & Payment System
* Admin Analytics Dashboard

---

## ⚙️ Setup Instructions

### Backend

cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend

cd frontend
npm install
npm start
